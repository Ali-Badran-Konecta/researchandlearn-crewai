# Product Overview

## Project Purpose
Researchandlearn is a multi-agent AI system powered by CrewAI that automates research and learning roadmap generation. The system leverages collaborative AI agents to conduct comprehensive research on any topic and produce structured learning materials.

## Key Features
- **Multi-Agent Collaboration**: Utilizes CrewAI framework for coordinated AI agent workflows
- **Automated Research**: Conducts thorough research on specified topics using AI agents
- **Learning Roadmap Generation**: Creates structured learning paths and educational content
- **Flexible Topic Input**: Accepts research topics via environment variables, command line, or interactive input
- **Markdown Output**: Generates well-formatted learning roadmaps in Markdown format
- **Training & Testing**: Supports crew training iterations and performance testing
- **Replay Functionality**: Can replay crew execution from specific tasks for debugging

## Target Users
- **Researchers**: Professionals needing comprehensive topic analysis and structured learning paths
- **Educators**: Teachers and trainers creating curriculum and learning materials
- **Students**: Learners seeking organized study guides and research summaries
- **Content Creators**: Writers and developers needing structured research for projects
- **Organizations**: Teams requiring standardized research processes and knowledge management

## Use Cases
- Generate learning roadmaps for emerging technologies (AI, LLMs, frameworks)
- Create structured research reports on complex topics
- Develop educational content and curriculum materials
- Automate knowledge discovery and organization processes
- Support decision-making with comprehensive topic analysis
- Build training materials for team development programs
